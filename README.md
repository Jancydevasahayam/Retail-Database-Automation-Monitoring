# Retail Database Automation & Monitoring

## Tech Stack
- Java
- JDBC
- MySQL / PostgreSQL
- Maven
- Git

## Project Overview
This project automates the ingestion of retail sales data from CSV files and REST APIs
into a relational database using Java JDBC. It includes data validation, optimized CRUD
operations, query performance monitoring, and automated database backup.

## Features
- CSV & API data ingestion
- Data validation and integrity checks
- CRUD operations using JDBC
- Query execution time monitoring
- Automated database backup
- Modular and scalable architecture
